    CLARION AI SMART BLACKBOARD

    Project Overview: Clarion AI Smart Blackboard is an AI-powered classroom assistant developed as a final-year major project. It enables gesture-based drawing, handwriting recognition, shape detection, and real-time AI explanations, creating an interactive teaching and learning environment. The project's technical contribution also resulted in a published research paper, demonstrating its academic impact and innovation.

    Status: Completed (Final Year Major Project)

    Key Features:

    • Gesture-based virtual whiteboard using MediaPipe Hand Tracking
    • Real-time freehand drawing and eraser functionality
    • Handwriting recognition using EasyOCR
    • Shape detection for circles, rectangles, squares, and triangles
    • AI-powered content analysis using Groq LLM (Llama 3.1)
    • Automatic explanation and summarization of board content
    • Session management with save and load functionality
    • PDF export with board snapshots and AI-generated notes
    • Keyboard and mouse support alongside gesture controls

    Technologies Used: Python, PyQt5, OpenCV, MediaPipe, EasyOCR, Groq API, ReportLab

    Libraries: OpenCV, MediaPipe, EasyOCR, NumPy, PyQt5, ReportLab

    Impact: Clarion bridges the gap between traditional teaching methods and AI-assisted learning by providing an intelligent classroom environment capable of understanding, analyzing, and explaining handwritten educational content in real time.

    Future Scope: Future enhancements include cloud synchronization, collaborative classrooms, multilingual support, adaptive learning assistance, smart note generation, and integration with Learning Management Systems (LMS).

    SMART ATTENDANCE SYSTEM

    Project Overview: The Smart Attendance System is an AI-based solution designed to modernize the traditional attendance process using computer vision and facial recognition. The application captures live facial images through a camera, detects and recognizes registered students using the K-Nearest Neighbors (KNN) algorithm, and automatically stores attendance records in an SQLite database. By eliminating manual roll calls, the system enhances efficiency, minimizes human errors, prevents proxy attendance, and provides a faster and more reliable attendance management solution for educational institutions.

    Status (In Progress):

    Pending: User login authentication on Android app or Streamlit.

    Technologies Used: Python, OpenCV, SQLite

    Libraries: OpenCV, Keras, Scikit-Learn

    Future Implementation: This system can be adopted in Indian educational institutions to streamline attendance and reduce errors. Future features could include multi-language support, parent notifications, and integration with student management systems.
